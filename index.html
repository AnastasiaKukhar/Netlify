and update working directory

List new or modified files not yet commited
&git status		 Created new branch called my_branch
&git branch new_branch		 Commit all staged files to versioned history
&git commit -an "commit message"		 Push local changes to origin
&git push
Show the changes to files not yet staged
&git diff		 Delete the branch called my_branch
&git branch -d my_branch		 Unstaged file, keeping the file changes
git reset [file]		 Finally!
